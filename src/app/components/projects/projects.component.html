Mis trabajos en Laboratoria

Data Lovers

Sitio web para fans de Rick and Morty, creado con Javascript vanilla, manipulando datos a través un archivo JSON. El usuario puede filtrar por categorías, como, personajes, estado, etc.

Proyecto realizado en Bootcamp Laboratoria Chile, desarrollado bajo metodologías agiles, aplicando dailys diariamente y ocupé las herramientas trello y slack.

Super Hambu

Super Hambu es un sistema web responsivo de comandas de comida rápida. Con servicios tanto para el mesero como para la cocina misma del local.

Proyecto realizado dentro del Bootcamp Laboratoria Chile, junto a Daniela Salermi. Al ser un proyecto colaborativo, trabajamos con control de versiones, GIT. Ocupamos herramientas como GitHub, Trello, Slack, y Heroku. Sistema desarrollado en Angular 8

Battle Zombie

Juego basado en el clasico BattleShip adaptado
a una version moderna, cambiando temática y funcionalidad. Se puede jugar en linea invitando a un segundo participante a través de un link.

Proyecto realizado bajo el framework de Angular 8, con herramientas como MongoDB, Heroku, GitHub.
